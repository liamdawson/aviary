# The Aviary

## Bootstrapping a Bird

```shell
curl -fsSL https://raw.githubusercontent.com/liamdawson/aviary/master/bootstrap.sh | sh
curl -fsSL https://raw.githubusercontent.com/liamdawson/aviary/master/run.sh | sh
```
